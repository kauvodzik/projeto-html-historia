História do HTML

1. O que é HTML? (<h1> a <h6>)

HTML significa HyperText Markup Language, ou Linguagem de Marcação de Hipertexto. Ele é usado para estruturar o conteúdo da web.

2. Origem do HTML (<p> e formatações)

O HTML foi criado por Tim Berners-Lee em 1991. Inicialmente, ele servia para compartilhar documentos científicos pela internet.

Hoje, é possível usar marcas como destacar texto, sublinhar partes importantes e até mostrar algo em tamanho menor.

Exemplo com sobrescrito: 102 = 100 | Subscrito: CO2

3. Evolução das versões (<ol> e <ul>)

Principais versões do HTML (ordenadas):

  1. HTML 1.0 - 1993
  2. HTML 2.0 - 1995
  3. HTML 4.01 - 1999
  4. HTML5 - 2014

Recursos comuns em versões HTML (não ordenada):

  • Formulários
  • Listas
  • Links
  • Imagens
